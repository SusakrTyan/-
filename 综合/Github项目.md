（该项目将语言模型聊天直接引入 Web 浏览器。一切都在浏览器中运行，没有服务器支持，并使用WebGPU加速。我们可以带来很多有趣的机会，为每个人构建AI助手，并在享受GPU加速的同时实现隐私。）
